🏡 Real Estate CRM — Frontend Team Project
A collaborative front‑end project built by Mako and Zach to practice real‑world development using HTML, Tailwind CSS, and JavaScript.
This project simulates a simple Real Estate CRM dashboard used by agents to manage clients, tasks, and notes.

Both developers will work with all three technologies, but each person will build different sections of the project to avoid conflicts and to practice independently.

📌 Project Purpose
The goal of this project is to:

Practice building real UI layouts

Practice Tailwind CSS styling

Practice JavaScript DOM manipulation

Practice GitHub collaboration

Practice dividing work like a real dev team

Build a functional CRM dashboard that can grow over time

This is a learning project, but it can evolve into a real SaaS product if we choose to expand it.

🛠️ Tech Stack
We are both practicing:

HTML — structure

Tailwind CSS — styling

JavaScript — interactivity

No frameworks. No back‑end.
Just pure front‑end fundamentals.

🔧 GitHub Workflow
1. Clone the repo
Code
git clone <repo-url>
2. Create your own branch
Each person works on their own branch:

Code
git checkout -b mako-section
git checkout -b zach-section
3. Work ONLY inside your assigned sections
Follow the HTML comments to know where to code.

4. Commit your changes
Code
git add .
git commit -m "Added navigation bar layout"
5. Push your branch
Code
git push origin mako-section
6. Open a Pull Request
Describe what you built

Tag the other person

Merge after review

This is how real dev teams collaborate.
